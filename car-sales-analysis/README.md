# Car Sales Exploratory Data Analysis (EDA)

This project performs an extensive exploratory analysis on a dataset of 50,000 car sales records to uncover market trends, pricing factors, and manufacturer benchmarks.

## 📊 Project Highlights
The analysis focuses on data wrangling and visualization to understand how vehicle specifications affect market value.

### Key Insights & Tasks:
* **Data Auditing:** Processed 50,000 records covering 39 years of manufacture (1984–2022).
* **Feature Engineering:** Computed new metrics such as "Price per Kilometer" to normalize value across different usage levels.
* **Extreme Value Analysis:** Identified market outliers, including the highest-priced vehicle (BMW M5) and lowest-mileage samples.
* **Comparative Analysis (Toyota vs. BMW):** * Conducted a side-by-side comparison of market positioning.
    * Found that while Toyota leads in volume and Hybrid technology, BMW commands nearly double the average revenue per unit.
* **Multivariate Visualization:** Used scatter plots and KDE histograms to map the relationship between Price, Mileage, and Fuel Type.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas (Data Manipulation), Seaborn & Matplotlib (Advanced Visualization).
* **Concepts:** Descriptive Statistics, Data Cleaning, Trend Analysis.
