# Advanced Data Programming & Statistical Analysis

This project demonstrates a comprehensive data science workflow using Python, developed as part of the MSc in Data Science curriculum at University College Dublin (UCD).

## 📊 Project Overview
The analysis explores complex datasets through statistical programming and machine learning to derive actionable insights.

### Key Components:
* **Exploratory Data Analysis (EDA):** Advanced visualizations using Seaborn and Matplotlib to identify distributions and correlations.
* **Statistical Modeling:** Implementation of Linear Regression models to analyze relationships between variables.
* **Classification & Predictive Analytics:** Application of classification algorithms to categorize data points based on feature sets.
* **Unsupervised Learning:** Cluster analysis using K-Means to identify underlying patterns and data segmentation.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn.
* **Environment:** Jupyter Notebook / Quarto.
